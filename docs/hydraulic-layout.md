# Hydraulic layout

The pressure sensor is added on a branch near the group/head pressure gauge side so the controller can react to real brew pressure.

![Hydraulic circuit layout](../images/hydraulic-circuit-layout.png)

The goal is not to replace the machine hydraulics, but to add pressure feedback to the existing pump control path.

Before opening fittings, let the machine cool down, unplug it from mains, and depressurize the coffee circuit.
