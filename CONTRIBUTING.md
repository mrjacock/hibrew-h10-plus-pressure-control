# Contributing

Contributions are welcome, especially:

- confirmed pinouts from other HiBREW H10A Plus revisions;
- pressure sensor calibration notes;
- safer mounting solutions;
- clearer diagrams;
- firmware improvements tested on real hardware;
- troubleshooting reports with photos and measurements.

## Safety-first contribution rule

Do not submit instructions that require unsafe live probing, exposed mains wiring, or USB connection while the machine is powered from mains.

## Good bug reports include

- machine model and production date if known;
- board revision if visible;
- Arduino model;
- pressure sensor model/range/output;
- wiring photos;
- pressure behavior;
- exact firmware parameters changed from default.
